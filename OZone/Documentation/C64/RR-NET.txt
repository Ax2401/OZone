Send Packet
	TxCMD = 00c0h
	TxLength = packet length
	Check BusST.BufferAvailable flag (PacketPage[0138h] & 0x0100 = 0x0100)
	Write even bytes to DataPort0
	Write odd bytes to DataPort0+1

Receive Packet
	Wait for RxEvent
	Read Event
		Read DataPort0+1
		Read DataPort0
	Read Packet Length
		Read DataPort0+1
		Read DataPort0
	ReadPacket
		Read DataPort0
		Read DataPort0+1
